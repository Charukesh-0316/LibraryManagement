# LibraryManagementAPI
CRUD Operations performed
-------------------------------------------------------------------

Endpoints
----------------------------------------------
      To get all books
- URL : /Books
- Method : Get
- Response : Status Code: 200(ok)

      To get specific Book
- URL : /id
- Method : Get
- Response : Status Code: 200(ok)

      To add book
- URL : /add_book
- Method : Post
- Response : Status Code: 200(ok)

      To update existing book
- URL : /update_book
- Method : PUT
- Response : Status Code: 200(ok)

      To delete book
- URL : /id
- Method : Delete
- Response : Status Code: 200(ok)

--------------------------------------------------------------------
Technology used :

- Java
- Spring Boot
- Hibernate/JPA
- REST API
